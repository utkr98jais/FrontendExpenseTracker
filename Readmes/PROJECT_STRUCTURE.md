## ğŸ¯ Key Files Explained

### Entry Points

- **`index.html`** - Static HTML template
- **`main.jsx`** - React app initialization with ExpenseProvider
- **`App.jsx`** - Route configuration

### Core Components

- **`ExpenseDashboard.jsx`** - Main dashboard (renders CategoryCards)
- **`CategoryCard.jsx`** - Category display with nested ExpenseItems
- **`CategoryForm.jsx`** - Modal for creating categories
- **`ExpenseForm.jsx`** - Modal for creating expenses

### State Management

- **`ExpenseContext.jsx`** - Global state (React Context)
- **`useExpense.js`** - Custom hook for accessing context
- **`ExpenseContext.backend.example.jsx`** - Template for API integration

### Configuration

- **`package.json`** - Project dependencies and scripts
- **`vite.config.js`** - Vite bundler configuration
- **`eslint.config.js`** - Code linting rules
- **`tailwind.config.js`** - (Generated by Tailwind) CSS framework config

### Documentation

- **`QUICK_START.md`** - Get started in 5 minutes
- **`FRONTEND_README.md`** - Complete feature documentation
- **`BACKEND_INTEGRATION.md`** - Connect to API backend
- **`COMPONENT_ARCHITECTURE.md`** - Component structure details
- **`IMPLEMENTATION_SUMMARY.md`** - What was built
- **`TESTING_CHECKLIST.md`** - How to test everything
